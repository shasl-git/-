\abstractRU{Предмет исследования – Совокупность методов, алгоритмов и программных решений для автоматизированного анализа данных о продажах и остатках товаров на маркетплейсах с использованием данных, полученных API методом, а также средств аутентификации и авторизации пользователей.
Изучить технологии и создать защищённое веб-приложение, которое:

1.	Получает отчёты через API методы маркетплейсов;

2.	Превращает сырые данные в структурированные наглядные показатели;

3.	Обеспечивает разграничение доступа между пользователями;

4.	Сохраняет данные о пользователях и их настройках.

Проект представляет собой аналитическую платформу для продавцов маркетплейсов, которая автоматизирует обработку данных, предоставляя комплексный анализ по продажам, остаткам, выручке и распределению по кластерам.
}
